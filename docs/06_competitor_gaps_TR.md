# Navex - Rakip Site İncelemesi & Entegrasyon Planı (10 site, canlı)

*10 rakibin canlı sitesi tek tek incelendi. Amaç: onlarda olan, bizde olmayan ve bizim siteyi
geliştirecek mekanikleri çıkarıp uygun olanları entegre etmek. Gerçekçilik kuralına (yeni butik,
"prior to Navex"), maritime-only ve no-em-dash kurallarına uyularak.*

---

## Bölüm 1 - Site site kısa çıkarım (bizde olmayan / güçlü mekanik)

- **Mercator** - her servis "VIEW PROJECTS" ile proof'a bağlı; sektör value-prop grid'i; "300+ engagements" rakamı; partner-imzalı kısa içerik.
- **Carter Morse** - "by the numbers" bandı; logolu tombstone'lar; **isim+ünvanlı testimonial carousel**; ekip ortalama deneyim ("25-year average").
- **AMA** - servisleri **deal tipine** ayırma (M&A / restructuring / capital raise / advisory); advisory + principal "skin in the game"; sektör özelliği vurgusu.
- **Marsoft** - **çatışmasız/bağımsız** konumlanma; **isimli metodoloji ("FLAGSHIP")**; case study = ihtiyaç→yöntem→ölçülebilir sonuç; stakeholder-bazlı sayfalar.
- **McLean** - sektör-grup sayfaları; PE-exit'e yazılmış içerik (QoE, earnout); "closed engagements since 2010" rakamı.
- **Poten** - content engine (Daily Briefing, imza raporlar, webinar); ton: *"advising with honesty even if it doesn't result in a transaction"*.
- **Pareto** - navy-clean estetik; **credential tile'ları** (logolu son işler); sektör haber hub'ı; yıllık konferans.
- **Fearnley** - ayrı **research portalı**; **ongoing transactions** takibi; **CEO mesajı**; çok-ofis global erişim.
- **Clarksons** - servisleri **birbirini besleyen ekosistem** olarak çerçeveleme; research'ü omurga yapma; şeffaf governance.
- **Drewry** - **ücretsiz index/tracker + premium** katman; servis hatlarını görsel ayırma; ölçülebilir müşteri başarı hikayeleri; haftalık/aylık ritim.

## Bölüm 3 - Entegrasyon adayları (öncelikli, kurallara uygun)

> **DURUM (2026-06-27): A, B, C, D entegre edildi.** A = Home lifecycle şeridi + servis→Case Studies
> bağı. B = Home "How we're different" bağımsızlık bölümü. C = Services "The Operating Bridge" isimli
> yaklaşım + About kurucu sözü. D = Insights featured yazı + newsletter. E/F beklemede (Harun verisi /
> ileriki data hub).

**A. Tam-yaşam-döngüsü şeridi + servis→case-study bağı** (Clarksons ekosistem + Mercator).
Diligence → Value Creation → Board/Interim → Exit readiness'i tek görsel continuum olarak göster;
her servisi Case Studies'e bağla. Wedge'imizi görselleştirir, "boşluğu biz dolduruyoruz" der.

**B. Bağımsızlık / dürüst-danışman bölümü** (Poten + Marsoft).
"Bizi başarı ücreti değil doğru karar yönlendirir; deal'i broker etmeyiz, veri satmayız, sizin
sonucunuza hizalıyız." Çatışmasız, üst-katman, butik güveni. Tek güçlü bölüm.

**C. İsimli yaklaşım + kurucu mesajı** (Marsoft FLAGSHIP + Fearnley CEO).
4 adımlı delivery approach'a bir isim ver (IP hissi); About'a Harun'dan 1 paragraf kısa kurucu sözü.

**D. Insights motoru yükseltmesi** (Poten/Drewry).
Newsletter kayıt alanı + "imza perspektif" çerçevesi + byline'lar; düzenli ritim sinyali.

**E. (İleri) Ücretsiz mini-tracker + premium** (Drewry) - Claude Code proprietary data hub ile ileride.
**F. (Harun gerekli) İsim/ünvanlı testimonial** - Carter Morse modeli.

> Uymayanlar (bilinçli atlanan): logolu tombstone/transaction tracker (yeni firma, gerçekçilik),
> FINRA rozeti (broker-dealer değiliz), global ofis ağı (tek ofis - ama "team has operated across
> NA/SA/EMEA/APAC" dürüst erişim sinyali olarak kullanılabilir).
