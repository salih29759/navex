# UTM Cheatsheet - Harun'un outreach linkleri

Kopyala-yapıştır hazır linkler. UTM'ler Vercel Web Analytics'te (navex projesi > Analytics >
UTM Parameters) otomatik görünür; hangi kanalın ziyaret ve lead getirdiğini oradan izlersin.

Not: Site statik ve hiçbir redirect query parametrelerini silmez (apex 308 -> www yönlendirmesi
parametreleri korur), UTM'ler güvenle kullanılabilir.

## Kendi linkini üretmek istersen
Şablon: `https://www.navexcapital.com/SAYFA?utm_source=KAYNAK&utm_medium=KANAL&utm_campaign=KAMPANYA`
- utm_source: linkedin, email, whatsapp, event ...
- utm_medium: dm, post, signature, newsletter ...
- utm_campaign: sponsor-outreach, owner-outreach, thought-leadership ...

Kural: hep küçük harf ve tire kullan; aynı kampanyada aynı yazımı koru (yoksa raporda ikiye bölünür).
